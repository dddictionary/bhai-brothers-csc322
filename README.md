# Run Instructions

- For frontend:
  - Navigate to `client` directory and run `npm i`
  - Then run `npm run dev` to run the frontend
- For backend:
  - Navigate to `backend` directory and start/setup your venv
  - Run `pip install -r requirements.txt`
  - Then run `python3 server.py`
